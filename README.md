# Power BI Data Analytics Course

Tôi đang học Power BI và thực hành các kỹ năng phân tích dữ liệu.

## Mục tiêu
- Học cách sử dụng Power BI để tạo báo cáo và biểu đồ
- Phân tích dữ liệu từ các nguồn khác nhau
- Xây dựng mô hình dữ liệu (Star Schema)

## Cấu trúc dự án
- `monthly_files/` - Dữ liệu hàng tháng
- `star_schema_files/` - Các bảng thuộc mô hình Star Schema
  - `company_dim.csv` - Bảng Dimension Công ty
  - `job_postings_fact.csv` - Bảng Fact Bài đăng công việc
  - `skills_dim.csv` - Bảng Dimension Kỹ năng
  - `skills_job_dim.csv` - Bảng liên kết Kỹ năng và Công việc

## Công cụ sử dụng
- Power BI Desktop
- CSV, xlsx files
